SeGIIR es un software desarrollado con el objetivo de realizar el seguimiento a la práctica Gestión Iterativa e Incremental de Requisitos de RUP (GIIR-RUP), la cual tiene como objetivo guiar la forma en la que se administran los requisitos en un proyecto de desarrollo de software.

Ademas de realizar el seguimiento a la práctica GIIR-RUP, el software proporciona toda la información de la práctica para poder aplicarla de forma adecuada. Se puede encontrar las actividades, tareas y el flujo de actividades en la pestaña de práctica.
